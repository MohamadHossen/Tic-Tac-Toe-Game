# Tic-Tac-Toe in C++

Console Tic-Tac-Toe game vs computer, written while learning C++.

## Features
- Player (O) vs Computer (X)
- Multiple rounds without interruptions
- Win/draw detection (rows/columns/diagonals)
- Clean board display
- Input validation

## What I practiced
- `struct` for game data
- Functions & pass-by-reference
- Arrays & game loops
- `rand()` for computer moves
- `cin` validation
- Win condition algorithms

## How to run
- Copy code to `main.cpp`
- Compile with any C++ compiler
- Run the executable in console
